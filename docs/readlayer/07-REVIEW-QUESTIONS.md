## 术语解释

### “索引对象”是什么

索引对象就是“一份拥有唯一 ID 的问题、答案或术语记录”。例如，一个术语不靠标题 `NPU` 连接，而靠 `termId=term_xxx` 连接。文章 A 和文章 C 的锚点都指向同一个 `termId`，因此修改该对象后两边都会显示最新内容；即使存在两个都叫 NPU 的对象，也不会串错。

### “旧手工块迁移”原来是什么意思

上一版是考虑把截图里已经手工排版的蓝色问答块和绿色术语块自动转成新索引对象。你确认目前不需要这件事，所以已从当前范围删除；旧内容保持原样即可。

### “学习画像”为什么删除

它原本指“语言、答案深度、是否举例”等偏好集合。由于你要求应用确定性且不自动学习，现在改为普通、显式、版本化的“回答规则/显示设置”，不再使用画像概念。
